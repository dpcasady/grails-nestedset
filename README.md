# Nestedset

[![Build
Status](https://travis-ci.org/ticketbis/grails-nestedset.png?branch=master)](https://travis-ci.org/ticketbis/grails-nestedset)

The nested set model is a particular technique for representing nested sets (also known as trees or hierarchies) in relational databases. This plugin
provides nestedset behaviour to domain classes.

## Installation

Add dependency to your BuildConfig;

```groovy
compile "com.ticketbis:nestedset:0.1.0"
```

## Usage

```groovy
// grails-app/domain
import com.ticketbis.nestedset

```
